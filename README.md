# game-1v1-2D

## Stack : 

*   Html

*   Css

*   Js


## The project

Simple turn by turn game, where you can use `slow attack` && `quick attack` to defeat your opponent, if you're HP are low, you can use the `heal` button !

But be carreful your mana isn't infinite.
